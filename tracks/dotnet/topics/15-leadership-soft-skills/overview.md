# Leadership & Soft Skills — Обзор

## Что это
Лидерство и soft skills — код-ревью, технические решения, менторинг, архитектурный ревью. Итоговый раздел, объединяющий технические знания с лидерскими навыками для Architect уровня.

## Подтемы
| # | Подтема | Описание |
|---|---------|----------|
| 1 | Code Review Culture | Культура код-ревью, чеклисты, обратная связь |
| 2 | Technical Decisions | ADR, trade-off analysis, RFC процесс |
| 3 | Mentoring | Менторинг, онбординг, growth plans |
| 4 | Architecture Review | Архитектурный ревью, fitness functions, ADR |
| 5 | Behavioral Interview | STAR метод, подготовка к поведенческим собесам |

## Порядок изучения
SOLID + Design Patterns → Code Review Culture → Mentoring
Clean Architecture + System Design → Technical Decisions
Microservices + Modular Monolith → Architecture Review
Behavioral Interview — независимая тема, можно изучать параллельно
