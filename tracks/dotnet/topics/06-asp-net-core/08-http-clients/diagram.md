# HTTP Clients & IHttpClientFactory — Диаграммы

## Архитектура IHttpClientFactory

```
┌─────────────────────────────────────────────┐
│              IHttpClientFactory              │
│                                             │
│  ┌─────────────┐    ┌─────────────┐         │
│  │  Handler A  │    │  Handler B  │  ...    │
│  │ (активен)   │    │ (активен)   │         │
│  │ создан 1мин │    │ создан 30с  │         │
│  └──────┬──────┘    └──────┬──────┘         │
└─────────┼─────────────────┼────────────────┘
          │                 │
   ┌──────┴──────┐   ┌──────┴──────┐
   │ HttpClient1 │   │ HttpClient2 │  ← новые на каждый запрос
   │  (лёгкий)  │   │  (лёгкий)  │
   └─────────────┘   └─────────────┘
```

## Цикл жизни HttpMessageHandler

```
t=0:00  ┌─────────────┐
        │  Handler A  │ ← создан, активен
        │             │ ← обрабатывает все запросы
t=2:00  │   EXPIRE    │ ← помечен как устаревший
        └─────────────┘   новые запросы → Handler B
              │
              │ (доделывает текущие запросы)
              ↓
        DISPOSE() → GC
```

## Состояния Circuit Breaker

```
         ┌──────────────────────────────────┐
         │                                  │
    ┌────▼────┐   5 ошибок подряд   ┌───────┴──────┐
    │ CLOSED  │ ──────────────────► │     OPEN     │
    │(работает│                     │(BrokenCircuit│
    │ нормально)                    │  Exception)  │
    └─────────┘                     └──────┬───────┘
         ▲                                 │ 30 сек
         │                          ┌──────▼───────┐
         │        успех             │  HALF-OPEN   │
         └──────────────────────────│ (1 пробный   │
              ошибка → OPEN         │   запрос)    │
                                    └──────────────┘
```

## Порядок политик Polly (цепочка)

```
Входящий запрос
       │
       ▼
┌─────────────────┐   5 провалов серий → BrokenCircuitException
│ Circuit Breaker │   Open → отклоняет мгновенно
└────────┬────────┘
         │
         ▼
┌─────────────────┐   При 5xx/таймауте → ждёт и повторяет
│     Retry       │   exponential backoff: 2s, 4s, 8s
└────────┬────────┘
         │
         ▼
┌─────────────────┐   10 секунд на попытку
│ Timeout(10s)    │   → TimeoutRejectedException
└────────┬────────┘
         │
         ▼
   HTTP запрос
```
