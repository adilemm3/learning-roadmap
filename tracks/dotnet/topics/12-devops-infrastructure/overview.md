# DevOps & Infrastructure — Обзор

## Что это
Инфраструктура — контейнеры, CI/CD, оркестрация, мониторинг, облако. Знания, необходимые для деплоя, масштабирования и поддержки .NET-приложений в продакшене.

## Подтемы
| # | Подтема | Описание |
|---|---------|----------|
| 1 | Docker & Containers | Dockerfile, docker-compose, multi-stage builds |
| 2 | CI/CD | GitHub Actions, Azure DevOps, пайплайны, стратегии деплоя |
| 3 | Kubernetes Basics | Pods, Services, Deployments, ConfigMaps, Helm |
| 4 | Monitoring & Logging | OpenTelemetry, Prometheus, Grafana, Serilog, ELK |
| 5 | Cloud: Azure & AWS | Основные сервисы Azure/AWS для .NET разработчика |

## Порядок изучения
Middleware Pipeline → Docker & Containers → CI/CD
Docker & Containers → Kubernetes Basics → Monitoring & Logging
Kubernetes Basics + CI/CD → Cloud: Azure & AWS

## Связи с другими разделами
Зависит от ASP.NET Core, Testing. Связан с System Design, Architecture (deployment).
