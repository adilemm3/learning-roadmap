## Сессия 2 — 2026-03-08

### Блок 3: Enums — внутреннее устройство

**Claude:** Аналогия: светофор — три состояния. Enum — именованные константы поверх числа. Под капотом всё равно число.

Enum — value type (System.Enum → System.ValueType → System.Object). Underlying type по умолчанию — int. Можно указать любой целочисленный тип (byte, short и т.д.) — для экономии памяти или совместимости с внешними API.

ToString/Parse/Cast:
- `light.ToString()` → "Green"
- `light.ToString("D")` → "2"
- `Enum.Parse<T>("Red")` → значение
- `(TrafficLight)2` → TrafficLight.Green
- `(TrafficLight)99` — НЕ бросает исключение! Компилируется и выводит "99". Для внешних данных — использовать Enum.IsDefined().

[Flags] — битовые маски для комбинации значений (права доступа: Read=1, Write=2, Execute=4). Значения — степени двойки (каждый бит = отдельный флаг). Комбинация через `|`, проверка через `&`. HasFlag() удобнее но медленнее (boxing). ToString с [Flags] возвращает "Read, Write" вместо "3".
