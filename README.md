# Learning Roadmap

Персональная система обучения технологиям с AI-ментором (Claude Code). Текущий фокус: .NET/C# — от Junior до Architect.

## Быстрый старт

```bash
git clone https://github.com/adilemm3/learning-roadmap.git
cd learning-roadmap
claude  # запустить Claude Code в директории проекта
```

Напиши `учим` — и обучение начнётся.

## Команды

### Обучение

| Команда | Что делает |
|---------|-----------|
| `учим` | Продолжить с места остановки |
| `учим [тема]` | Начать конкретную тему (`учим gc`, `учим linq`, `учим #3`) |
| `дальше` | Следующий блок |
| `углубись` | Подробнее о текущем моменте |
| `практика` | Задачи по текущей теме |
| `пропусти` | Пропустить блок |
| `стоп` | Сохранить и завершить сессию |

### Подготовка к собеседованию

| Команда | Что делает |
|---------|-----------|
| `вопросы [тема]` | Вопросы по уровням (Junior → Architect) |
| `собес [уровень]` | Mock-интервью |
| `собес system-design` | System Design интервью |
| `шпаргалка [тема]` | Шпаргалка для чтения с телефона (2 мин) |
| `карточки [тема]` | Флэш-карточки для повторения |

### Аналитика

| Команда | Что делает |
|---------|-----------|
| `прогресс` | Таблица прогресса |
| `слабые места` | Пробелы и план их закрытия |
| `что дальше` | Следующая тема по roadmap |
| `оцени меня [тема]` | Оценка уровня через вопросы |

### Прочее

| Команда | Что делает |
|---------|-----------|
| `кейс [тема]` | Реальный кейс или постмортем |
| `схема [тема]` | Визуализация (ASCII/mermaid) |
| `запуши` | Git push |

## Как устроено обучение

1. **Калибровка** — Claude задаёт 2-3 вопроса чтобы понять твой уровень
2. **План** — тема разбивается на 3-7 блоков
3. **Цикл** — для каждого блока: аналогия → теория → код → проверка → фидбек
4. **Прогресс** — сохраняется автоматически, можно продолжить в любой момент

## Хочешь свою копию?

### Предварительно: что нужно

- [Claude Code](https://claude.ai/code) — установленный и залогиненный
- Git

### Шаг 1 — Создай свой репозиторий

Нажми кнопку **«Use this template»** вверху страницы → **«Create a new repository»**.

> Если кнопки нет — перейди на ветку [`template`](../../tree/template) и нажми там.

### Шаг 2 — Клонируй и настрой

```bash
git clone https://github.com/ВАШ_USERNAME/learning-roadmap.git
cd learning-roadmap
bash scripts/personalize.sh
```

Скрипт спросит твоё имя и GitHub username — это займёт 30 секунд. Прогресса нет, всё чисто.

### Шаг 3 — Запусти и начни учиться

```bash
claude
```

Claude поприветствует тебя и предложит начать. Напиши `учим` — и поехали.

---

> **Примечание:** если хочешь изучать не .NET/C#, а другую технологию — после запуска скажи Claude: `добавь трек golang` (или любую другую). Структура создастся автоматически.
